# Salla-Project
# اسم المشروع (مثال: تقرير تحليل مبيعات شركة تجارة إلكترونية)

## 📌 نبذة عن المشروع
تحليل شامل لبيانات [اكتب هنا نوع البيانات، مثل: المبيعات / الموارد البشرية] باستخدام **Excel** و **Power BI** لتحديد نقاط القوة، ومعرفة المنتجات الأكثر مبيعاً، وتقديم توصيات لزيادة الأرباح.

## 🛠️ الأدوات المستخدمة (Tools Used)
*   **Microsoft Excel**: لتجميع البيانات الخام، وتنظيفها الأولي.
*   **Power Query**: داخل Power BI لتنظيف البيانات المتقدم (Data Transformation) وتغيير أنواع البيانات.
*   **DAX (Data Analysis Expressions)**: لإنشاء المقاييس الذكية (Measures) والأعمدة المحسوبة لحساب نسب النمو والأرباح.
*   **Power BI Desktop**: لتصميم لوحة تحكم تفاعلية وديناميكية (Interactive Dashboard).

## 🚀 خطوات العمل (Methodology)
1.  **استخراج البيانات وتنظيفها (ETL):** استيراد الملف من Excel، وإزالة القيم الفارغة والتكرارات عبر Power Query.
2.  **نمذجة البيانات (Data Modeling):** ربط الجداول ببعضها (مثل جدول المبيعات بجدول الموظفين أو الفروع).
3.  **كتابة معادلات DAX:** حساب [اذكر مقياسين مثل: إجمالي الأرباح، نسبة الزيادة السنوية].
4.  **التصور البصري (Data Visualization):** تصميم المخططات البيانية وتوزيع الفلاتر (Slicers) لتسهيل القراءة.
